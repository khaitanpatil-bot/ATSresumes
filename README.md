# Resume ATS Scanner UI

A React + Tailwind CSS landing page scaffold for an ATS resume scanner product with Supabase backend.

## Project Structure

- `frontend/` - React frontend application
- `backend/` - Backend services (Supabase integration)

## Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```

## Backend

This project uses Supabase as the backend for authentication and data storage. Configure your Supabase project URL and anon key in `frontend/.env`.

## Project structure (Frontend)

- `src/` - React components and styles
- `tailwind.config.js` - Tailwind configuration
- `vite.config.js` - Vite project setup
- `index.html` - app entrypoint
